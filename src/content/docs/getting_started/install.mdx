## MacOS

### Community Edition

<FileTree>
- fonts
  - color
    - **VF-Canto.ttf**
- starters/
- stylebot/
</FileTree>

<Steps>

1. Unzip the downloaded file
2. Double-click `VF-Canto.ttf` in the `fonts/color/` folder
3. Click *Install*

</Steps>
